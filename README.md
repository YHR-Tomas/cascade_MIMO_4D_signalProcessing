# Cascade MIMO 4D Signal Processing

This repository contains a MATLAB processing script for cascaded mmWave MIMO radar data. The code builds a 4D radar representation from raw ADC data captured by a cascaded TI mmWave radar system and converts the processed radar response into both polar and Cartesian radar intensity cubes.

The main processing flow is:

```text
Raw ADC data
      ↓
Cascade calibration
      ↓
Range FFT
      ↓
Doppler FFT
      ↓
Virtual antenna array formation
      ↓
2D azimuth/elevation beamforming
      ↓
Doppler compression
      ↓
Polar radar cube
      ↓
Cartesian voxel grid
```

## Required TI Folder and Configuration Setup

This repository is designed to work together with the TI **mmWave Studio 03.00.00.14** cascade MIMO MATLAB processing chain.

This code is **not fully standalone**. It depends on the TI cascade MIMO MATLAB example libraries, including calibration, range processing, Doppler processing, DOA processing, and JSON parameter generation functions.

The required TI library is usually located at:

```text
C:\ti\mmwave_studio_03_00_00_14\mmWaveStudio\MatlabExamples\4chip_cascade_MIMO_example\main\cascade
```

The script depends on TI cascade processing functions such as:
```text
calibrationCascade
rangeProcCascade
DopplerProcClutterRemove
DOACascade
JsonParser
getBinFileNames_withIdx
getValidNumFrames
add_paths.m
module_param.m
testList.txt
```
Therefore, before running this repository, the TI mmWave Studio cascade MATLAB example must be installed and configured correctly.

## Configuration Files

This project depends on both the radar capture configuration files generated by **TI mmWave Studio** and the MATLAB parameter files used by the **4-chip cascade MIMO processing chain**.

A typical capture folder from mmWave Studio may contain configuration files such as:

```text
Cascade_Capture_22xx.mmwave.json
Cascade_Capture_22xx.setup.json
```
